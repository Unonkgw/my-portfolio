KohiiBara

A Cafe Website UI Design Made in Figma

KohiiBara is a cozy, anime-inspired café website UI designed in Figma, featuring a warm aesthetic and an interactive prototype that showcases an intuitive menu navigation and smooth user flow. It blends playful visuals with a clean layout to enhance the online ordering experience.
